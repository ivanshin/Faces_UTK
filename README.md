
# Faces 👱‍♂️ 👩‍🦳 🧑‍🦳 👨‍🦳

This project is dedicated to convolutional models and working with face portraits.
As part of it, several convolutional networks were trained:
  1) Determination of age
  2) Sex (binary)
  3) Face detection


## Tech Stack

Gradio, Tensorflow
